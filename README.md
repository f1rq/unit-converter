## Installation
| OS | Format | Instructions |
| -- | ------ |--------------------------------------------------------------------|
| Windows | [`.exe`](https://github.com/f1rq/unit-converter/releases/latest) | Download and run. |
| Linux | [`.AppImage`](https://github.com/f1rq/unit-converter/releases/latest) | `chmod +x` and execute. |
| macOS | [`.zip`](https://github.com/f1rq/unit-converter/releases/latest) | Extract and open `.app`. |

---

**Linux Guide**

To run the **AppImage**, you can use the terminal:
```bash
$ chmod +x UnitConverter_v0.3.0-linux_x64.AppImage
$ ./UnitConverter_v0.3.0-linux_x64.AppImage
```
Or double-click the file in your file manager (may require `chmod +x` first).

**macOS Guide**

<details>
<summary><b>Disclaimer for macOS users!</b></summary>
These builds (arm64 & x64) are currently **untested** as I do not have access to Apple hardware. If you encounter any issues, please report them in the [Issues](https://github.com/f1rq/unit-converter/issues) section.
</details>

---

Since the app is not signed, you might see a security warning. To open it:
1. **Right-click** (or Control-click) the app icon.
2. Choose **Open** from the shortcut menu.
3. Click **Open** again in the dialog box.
